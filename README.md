# AI FAQ Chatbot

## Description
An AI-powered FAQ Chatbot developed using Python, NLP, TF-IDF Vectorization, and Cosine Similarity.

## Features
- FAQ-based Question Answering
- NLP Text Preprocessing
- TF-IDF Vectorization
- Cosine Similarity Matching
- Confidence Score
- CSV Dataset Support

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- NLTK

## How to Run
1. Install requirements
2. Run the notebook
3. Ask questions through the chatbot interface
